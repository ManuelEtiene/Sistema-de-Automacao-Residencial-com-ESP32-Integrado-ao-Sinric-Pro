# Documentação de Software e Instalação

## 💻 Ambiente de Desenvolvimento e Ferramentas
* **Firmware:** Arduino IDE (Versão 2.x ou superior recomendada)
* **Backend:** Node.js (Versão LTS) & Express Framework
* **Banco de Dados:** MongoDB Atlas ou Instância Local

## 📚 Bibliotecas Utilizadas (Firmware ESP32)
Para compilar o código com sucesso na Arduino IDE, instale as seguintes bibliotecas através do Gerenciador de Bibliotecas:
1. `Adafruit BMP280 Library` (Leitura do Sensor)
2. `PubSubClient` por Nick O'Leary (Comunicação de mensageria MQTT)
3. `ArduinoJson` por Benoit Blanchon (Estruturação dos payloads de telemetria)
4. `LiquidCrystal_I2C` (Controle do display local)
