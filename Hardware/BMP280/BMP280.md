
# Sensor de Pressão e Temperatura - BMP280

O sensor BMP280 é utilizado no sistema para realizar leituras contínuas de temperatura ambiente, fornecendo os dados necessários para o algoritmo de controle por histerese do Ar-Condicionado.

## Dados Técnicos do Sensor BMP280

| Característica | Descrição |
| :--- | :--- |
| **Tipo** | Sensor digital de pressão barométrica e temperatura |
| **Tensão de Operação** | 1.71V a 3.6V (Operando em 3.3V) |
| **Interface de Comunicação** | I2C ou SPI (Utilizado em modo I2C) |
| **Faixa de Temperatura** | -40°C a 85°C |
| **Precisão de Temperatura** | ±1.0°C |
| **Consumo de Corrente** | 2.7 µA a 1 Hz |

## Pinagem do BMP280

| Pino | Função | Descrição |
| :---: | :---: | :--- |
| 1 | VCC | Alimentação positiva conectada ao pino 3.3V do ESP32 |
| 2 | GND | Referência de terra conectada ao GND do ESP32 |
| 3 | SCL | Linha de clock do barramento I2C (conectada à GPIO 22) |
| 4 | SDA | Linha de dados do barramento I2C (conectada à GPIO 21) |

## Componentes Associados
* Módulo sensor BMP280 (Placa de circuito GY-BM)
* Cabos de conexão do tipo Jumpers fêmea-fêmea

## Recomendações
* **Endereço I2C:** Certifique-se de usar o endereço correto no código (`0x76` ou `0x77`), dependendo do fabricante do seu módulo.
* **Alimentação:** Nunca ligue o pino VCC na saída 5V (VIN) do ESP32, pois isso pode queimar o sensor de forma permanente.
* **Isolamento:** Evite posicionar o sensor muito próximo ao chip do ESP32 para que o calor gerado pelo processador e pelo Wi-Fi não mascare a leitura real do ambiente.

## Exemplo de Código

Segue o código de teste utilizado para a leitura isolada de dados de temperatura do BMP280 usando a Arduino IDE:

```cpp
#include <Wire.h>
#include <Adafruit_BMP280.h>

Adafruit_BMP280 bmp; // Cria a instância do sensor

void setup() {
  Serial.begin(115200);
  Serial.println("Iniciando leitura do BMP280...");

  // Inicializa o sensor com o endereço I2C padrão 0x76
  if (!bmp.begin(0x76)) {
    Serial.println("Falha ao encontrar o sensor BMP280!");
    while (1);
  }
}

void loop() {
  float temperatura = bmp.readTemperature();
  
  if (isnan(temperatura)) {
    Serial.println("Falha ao ler os dados do sensor!");
  } else {
    Serial.print("Temperatura: ");
    Serial.print(temperatura);
    Serial.println(" °C");
  }

  delay(2000); // Intervalo mínimo recomendado entre leituras
}
